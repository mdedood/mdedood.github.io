Introduction

This online course is designed as part of the Bachelor Research Skills course for third year physics students at Leiden university. The purpose of this course is to prepare students to the use of data analysis methods that they will need during their research project. Students should have a working knowledge of Python and he use of Jupyter notebooks. The focus of this course is on the interpretation of fit results, error bars and possible pitfalls and difficulties that are frequently encountered during experimental research. I assume that students have already used a computer to visualize their data and have done non-linear least square fitting of data using a standard python library, e.g. SciPy.

How to use this course?

This course consists of two modules that each contain a part A and part B. Each part is designed to take 1-1.5 hour to complete and contains a combination of quizzes using Google forms, some instruction videos and a set of Jupyter notebooks with detailed instructions, lecture notes etc. In principle these modules can be completed individually and independently, but discussion in small groups of 2-5 students is highly recommended.


For physics students at Leiden university live sessions are planned for each part. These sessions will consist of 30 min. of interactive lecture/discussion on part A of each module. Short instructions and a question and answer session is available for the B part of the module afterwards. Students are expected to come prepared by completing the A part of each module. The B part of each module contains at least one exercise that needs to be handed in.

Module content and learning goals

Module 1A // Data analysis, mistakes and misconduct

After completing this module the student:
- is able to name and explain ethical guidelines for conducting scientific research;
- is able to distinguish between scientific mistakes and scientific misconduct;
- can give a few examples of scientific mistakes and misconduct and discuss these with others


Module 1B // Errors, hypothesis testing and goodness of fit

In this module students learn to the statisitical methods of hypothesis testsing.
More specifically, the student can:
- calculate the chi-squared value and use this to test a hypothesis;
- asses the quality of a fit using the chi-squared dsitribution;
- dicuss the differences between various methods to fit a straight line


Module 2A // Non-linear-least-squares fitting

This module contains three examples that teach the student to:
- apply non-linear-least-squares fitting in different settings;
- use the method of error propagation to calculate the error bar on a data point and discuss the consequences for non-linear-least-squares fitting;
- understand the importance of initial parameters and find the overall best fit parameters;


Module 2B // Global warming in the Netherlands?

The goal of this module is to analyze a large online dataset using Pandas.
After completion the student is able to:
- use Pandas (in Python) on large online datasets.
- have an informed opinion whether or not the rise in temperature in the Netherlands over the past century is statistically significant.
- apply all the skills from all above modules to complete this final exercise.
